Installazione

È possibile estrarre l'archivio ZIP in una cartella qualsiasi. Poi creare una scorciatoia - trascinare il file EXE con il tasto destro del mouse e rilasciarlo sul desktop o sul pulsante Start.

Disinstallazione

Scegliere "Elimina Impostazioni" dal menu File. Quindi rimuovere i file di programma dal disco.

Esempi

Il file examples.cal contiene alcune macro utili. Salvare le proprie macro su un altro file, altrimenti saranno sovrascritte al successivo aggiornamento della nuova versione.

Costanti ed unità

I file constants.cns, units.unt contengono i menu delle Costanti ed Unità. È possibile modificare questi file. Si dovrebbe fare il backup di tutte le modifiche, perché questi file verranno sovrascritti al successivo aggiornamento della nuova versione.
