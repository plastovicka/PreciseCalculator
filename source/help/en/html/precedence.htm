Precedence of operators

Parenthesis are used to determine an order of evaluation. The calculator automatically appends missing right parenthesis at the end of a command.

Operators in expressions without parenthesis are executed in the following order:

  1. prefix operators (except -) if their operand is in parenthesis
  2. postfix operators
  3. #
  4. ^, **
  5. prefix operators if their operand is not in parenthesis
  6. combin, permut, nCr, nPr
  7. vert
  8. *, /, div, mod
  9. +, -
  10. lsh, rsh, rshi, <<, >>, >>>
  11. >, <, >=, <=
  12. ==, !=, <>
  13. and, &, nand, bitnand
  14. xor
  15. or, |, nor, bitnor
  16. imp, ->, bitimp
  17. eqv, biteqv
  18. =
  19. ,
  20. \
Examples

sin(30)^2 = (sin 30)^2 = 0.25
sin30^2 = sin(30^2) = sin900 = 0
2^3^4 = 2^(3^4) = 2^81
3*4+5*2^3 = (3*4)+(5*(2^3)) = 12+5*8
2+5 combin 3+1 = 2+(5 combin 3)+1
2+5 << 3+1 = (2+5)<<(3+1)
