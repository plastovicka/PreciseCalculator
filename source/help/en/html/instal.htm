Installation

You can extract ZIP archive to any folder. Then create a shortcut – drag EXE file with the right mouse button and drop it on the Desktop or on the taskbar.

Uninstallation

Choose "Delete settings" from the File menu. Then remove program files from your disk.

Examples

File examples.cal contains some useful macros. Save your own macros to some another file, otherwise they will be overwritten on the next upgrade to a new version.

Constants and Units

The files constants.cns, units.unt contain menus Constants and Units. You can modify these files. You should backup all your changes, because these files will be overwritten on the next upgrade to a new version.
