Computation time

n means precision, m means length of an input parameter

  • (const) : trunc, real, conjg, count, imp, eqv, nand, nor

  • (n) : +, -, and,or,xor,not,bitnand,bitnor,bitimp,biteqv, lsh,rsh,rshi, ctof,ftoc, gradtodeg,degtograd, dms,todms, min,max, med,mode, sum,ave, round,floor,ceil,frac, complex,imag

  • (n^1.585) : *, degtorad, gradtorad, rand,random, sumq,aveq,var,vara

  • (n^2) : /, div, mod, sqrt, hypot, abs, sign, fibon, radtodeg,radtograd, stdev,stdeva, print

  • (less than n^2.585) : exp, sin,cos,tan, cosec,sec,cotg, sinh,cosh,tanh, csch,sech,cotgh, polar

  • (n^2.585) : ^, #, ln,log,logn, asin,acos,atan, acsc,asec,acotg, asinh,acosh,atanh, acsch,asech,acotgh, angle, arg

  • (n^2 to n^3) : pi

  • (m^3) : gcd, lcm

  • (exp m) : prime, isprime, divisor, !, !!, combin, permut

Multiplicative constants were omitted. Computation time does not depend only on a precision, but often on a function parameter, too. Any number can be multiplicated or divided by a fraction in time (n).
